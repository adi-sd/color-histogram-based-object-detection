How to Run My Assignment 1 Code

( I have used JDK 20.0.2 )

1. Unzip assignment-2-submission.zip

2. Go to the unzipped folder

3. it will be a maven project

4. Use the following command to compile the Code
$ mvn clean package

6. To run the application

    1. form the project root or where the JAR is
    $  java -jar assignment-2-1.0-SNAPSHOT.jar <input_image_path> <obj1> <obj2> .....
